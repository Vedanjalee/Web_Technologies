HTML
HTML (HyperText Markup Language) is the standard language used to create and structure web pages. It defines the content of a webpage using elements such as headings, paragraphs, images, links, tables, and forms. HTML uses tags
CSS
CSS is a styling language used to control the appearance and layout of HTML elements on a webpage. It allows developers to apply colors, fonts, spacing, animations, and responsive designs to create visually appealing websites.
JS
JavaScript is a programming language used to make web pages interactive and dynamic. It enables functionalities like event handling, form validation, animations, API calls, and DOM manipulation.
